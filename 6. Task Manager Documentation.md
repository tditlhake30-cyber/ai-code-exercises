# 1. Project Information Chosen

**Project name:** Task Manager

**Description:**  
A Python based task management application that allows users to create tasks, update task status, set priorities, assign due dates, and organize tasks with tags.

**Key features:**
- Create and manage tasks
- Update task status and priority
- Set due dates
- Add and remove tags
- View overdue tasks
- Generate task statistics
- Store tasks in a JSON file

**Technologies used:**
- Python
- JSON storage
- argparse
- datetime

**Basic project structure:**
- `app.py` – main task management logic
- `models.py` – task model and enums
- `storage.py` – handles saving and loading tasks
- `cli.py` – command line interface
- `tasks.json` – stores task data
