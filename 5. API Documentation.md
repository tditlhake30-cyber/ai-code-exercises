## Overview

# 1. Original API Endpoint Code

```python
@app.route('/api/users/register', methods=['POST'])
def register_user():
    """Register a new user"""
    data = request.get_json()

    required_fields = ['username', 'email', 'password']
    for field in required_fields:
        if field not in data:
            return jsonify({
                'error': 'Missing required field',
                'message': f'{field} is required'
            }), 400

    if User.query.filter_by(username=data['username']).first():
        return jsonify({
            'error': 'Username taken',
            'message': 'Username is already in use'
        }), 409

    if User.query.filter_by(email=data['email']).first():
        return jsonify({
            'error': 'Email exists',
            'message': 'An account with this email already exists'
        }), 409

    if not re.match(r"^[^@]+@[^@]+\.[^@]+$", data['email']):
        return jsonify({
            'error': 'Invalid email',
            'message': 'Please provide a valid email address'
        }), 400

    if len(data['password']) < 8:
        return jsonify({
            'error': 'Weak password',
            'message': 'Password must be at least 8 characters long'
        }), 400

    password_hash = generate_password_hash(data['password'])

    new_user = User(
        username=data['username'],
        email=data['email'].lower(),
        password_hash=password_hash,
        created_at=datetime.utcnow(),
        role='user'
    )

    db.session.add(new_user)
    db.session.commit()

    user_data = {
        'id': new_user.id,
        'username': new_user.username,
        'email': new_user.email,
        'created_at': new_user.created_at.isoformat(),
        'role': new_user.role
    }

    return jsonify({
        'message': 'User registered successfully',
        'user': user_data
    }), 201
```

# 2. Endpoint Documentation (Prompt 1)

## Register User API

Create a new user account in the system.

### Endpoint

POST /api/users/register

### Description

This endpoint allows a new user to create an account by submitting a username, email address, and password.  
The API validates the input data, ensures the username and email are unique, and stores the user in the database.

### Request Body

| Field | Type | Required | Description |
|-----|-----|-----|-----|
| username | string | Yes | Unique username for the user |
| email | string | Yes | User email address |
| password | string | Yes | Password (must be at least 8 characters) |

### Example Request

```json
POST /api/users/register
Content-Type: application/json

{
  "username": "john123",
  "email": "john@example.com",
  "password": "mypassword123"
}
```

### Success Response

Status Code: **201 Created**

```json
{
  "message": "User registered successfully",
  "user": {
    "id": 1,
    "username": "john123",
    "email": "john@example.com",
    "created_at": "2026-03-10T10:30:00",
    "role": "user"
  }
}
```

### Error Responses

| Status | Error | Description |
|------|------|------|
| 400 | Missing required field | A required field was not included |
| 400 | Invalid email | Email format is incorrect |
| 400 | Weak password | Password shorter than 8 characters |
| 409 | Username taken | Username already exists |
| 409 | Email exists | Email already registered |
| 500 | Server error | Internal server error occurred |

### Authentication

No authentication required for registration.

# 4. Developer Usage Guide (Prompt 3)

## How to Register a User

This guide explains how developers can create a user using the registration API.

### Step 1 – Prepare the Request

Send a **POST request** to:

```
/api/users/register
```

The request must include a JSON body with:

- username
- email
- password

Example:

```json
{
  "username": "newuser",
  "email": "newuser@example.com",
  "password": "securepassword123"
}
```

### Step 2 – Send the Request

Example using Python:

```python
import requests

url = "http://localhost:5000/api/users/register"

data = {
    "username": "newuser",
    "email": "newuser@example.com",
    "password": "securepassword123"
}

response = requests.post(url, json=data)

print(response.json())
```

### Step 3 – Handle the Response

If successful, the API returns:

- confirmation message
- user information

Example response:

```json
{
  "message": "User registered successfully",
  "user": {
    "id": 1,
    "username": "newuser",
    "email": "newuser@example.com"
  }
}
```

### Handling Errors

Common errors include:

- missing required fields
- invalid email format
- password too short
- username already taken

Developers should check the **HTTP status code** and display the error message returned by the API.
